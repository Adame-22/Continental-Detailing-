# Project Blueprint

## Overview

This project is a web application that has been reset to a clean slate. It now serves as a starting point for new development.

## Project Structure

- `public/`: This directory contains all the static assets for the project.
  - `index.html`: The main HTML file.
  - `style.css`: The main stylesheet.
  - `main.js`: The main JavaScript file.
- `firebase.json`: The Firebase configuration file, pointing to the `public` directory.
- `blueprint.md`: This file, documenting the project structure and changes.

## Application Details

- **Frameworks:** None. This is a vanilla HTML, CSS, and JavaScript project.
- **Functionality:** A simple heading is displayed.
